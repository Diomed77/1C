   Значение константы "ОсновнаяВалютаЗакупки" используется в качестве значения 
по умолчанию для реквизита "Валюта" для новых приходных накладных.   
Значение константы устанавливается путем выбора валюты из  справочника "Валюты". 
Установить значение константы можно так-же при помощи обработки "Настройка параметров 
торговли" 
   В качестве значения константы рекомендуется выбрать валюту, наиболее часто 
используемую в качестве валюты приходных накладных.
