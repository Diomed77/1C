    «ОсновнаяСтавкаНДС» — это значение ставки НДС (в процентах) по умолча-нию для товаров. Эта Константа может
принимать только одно из следующих значений "безНДС", "20%", "10%". (Это сделано в связи с тем, что
в документах "Счет_фактура",  должны содержать суммы по НДС только таких значений.) Это значение
будет по умолчанию подставляться в поле «СтавкаНДС» при вводе нового товара в справочник «Товары».
Здесь следует установить значение ставки НДС, наиболее часто ис-пользуемое для товаров.
Например, установите значение "20%".
