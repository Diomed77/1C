Справочник "Настройка складов". 

Содержит дополнительные сведения о складах. Сведения собраны в
отдельном справочнике, для того чтобы не вносить изменения в 
стандартный справочник "Склады" конфигурации, для удобства обновления
конфигурации.       
Использование данного справочника имеет смысл только при использовании
ПО Агент+ Инвент, т.е. при "торговле с колес".
Первоначальное заполнение справочника производится автоматически
при первой выгрузке данных в МУ. При автоматическом создании новых 
элементов справочника заполняются реквизиты "Ссылка", "Идентификатор"
поэтому для корректности обмена данными данные реквизиты следует не 
изменять.
Справочник содержит следующие сведения:
• Ссылка. Элемент справочника "Склады";
• Идентификатор. Идентификатор, используемый для обмена данными;
• Выгружать в КПК. Признак выгрузки данного склада в КПК;
• Передвижной. Признак, что склад является передвижным (носит 
  информационный характер);
• Тех осмотр. Дата тех. осмотра (носит информационный характер);
• Тоннаж. Грузоподъемность автомобиля (носит информационный характер).

В КПК выгружаются только те склады, которые отмечены признаком "Выгружать".
