   Справочник "Товары" в типовой конфигурации реализован трехуровневым, т.е. для товаров можно
завести группы и подгруппы товаров. Товар имеет четыре цены: три из них - цены
реализации: розничная цена, мелкооптовая цена, оптовая цена.
   Товар имеет четыре цены: приходную, розничную, мелкооптовую, оптовую. Предполагается, что в
справочнике товаров всегда хранят-ся значения цен на товары с учетом НДС. Цены товара
используется как цены по умолчанию для первоначальной подстановки в графу «Цена» при
выписке счетов и накладных. 
   Реквизит товара "СтавкаНДС" может принимать только одно из следующих
значений "безНДС", "20%", "10%".
    Реквизит товара "ВидТовара" может принимать одно из следующих
значений: "Товар", "Набор", "Услуга".
    Реквизиты товара: "СтоимостьСборки", "СтоимостьРазборки", "ВалютаРаботы"
необходимы и имеют смысл только для товаров вида "Набор".
   Реквизит товара "ПорядокСписания" необходим и имеет смысл только для товаров
вида "Услуга". Этот реквизит может принимать только одно из следующих
значений: "Полностью", "Пропорционально".
    Приходная цена товара используется как цена по умолчанию для первоначальной подстановки
в графу "Цена" при выписке закупочных накладных. При описании товара можно указать три значения
валюты: валюту реализации, валюту закупки и валюту учета товара. Эти значения учитываются
при пересчетах цен и стоимостей товаров.
       Внимание!!! Выбор валюты учета товара - ответственное решение, которое надо принимать
до начала работы с товаром. Изменять значение валюты учета товара в процессе эксплуатации
системы нельзя, т.к. это приведет к неверным значениям остатков в Регистрах (после этого
придется делать полное перепроведение всех Документов).
	Кроме того, товар характеризуется базовой единицей измерения. Базовая единица
измерения - это единица, в которой ведется учет наличия товара.
	Кроме базовой единицы измерения товар может иметь много других единиц измерения.
Например: сигареты могут иметь следующие единицы: пачка, блок, коробка, контейнер, вагон.
Каждая единица измерения товара имеет свой коэффициент пересчета к базовой единице.
Таким образом, в типовой конфигурации товары можно выписывать в документах в любой
единице измерения, которую завели для данного товара. Описания единиц измерения для товаров
ведется в подчиненном Справочнике "Единицы". 
   Справочник "Комплекты" подчинен справочнику "Товары". Этот справочник используется и
имеет смысл только для товаров вида "Набор". В этот справочник заносятся состав
комплектующих для Товаров-комплектов. 
   В печатных формах документов при печати строки спецификации с товаром вида "Набор"
дополнительно распечатывается состав комплекта.
   	Окно списка товаров имеет дополнительный элемент управления - кнопку "Печать".
По  нажатию этой кнопки формируется стандартный табличный документ СИСТЕМЫ "1С:ТОРГОВЛЯ" ,
содержащий каталог товаров, который можно просмотреть и распечатать. Каталог товаров можно
выводить полностью или частично. Если курсор в списке товаров указывает на группу справочника
верхнего уровня, то каталог товаров будет выведен полностью. Если курсор указывает на
подгруппу или элемент (товар) справочника, то будет сформирован только один раздел
каталога товаров, которому принадлежит выбранный курсором элемент справочника.
          В перечисление "ВидыТоваров" добавлены два новых вида: Блюдо и Ингредиент.
При выборе в реквизит справочника "Вид товара" - "блюдо" Появляется закладка "Блюдо" со следующими реквизитами:
- Количество калькуляции - количество единиц блюда, на которое составляется калькуляция.
- Единица калькуляции - единица измерения, в которой указывается количество калькуляции.
- Описание для меню - поясняющий текст произвольного размера, который потом будет использоваться в печатной форме меню. 

При выборе в реквизит справочника "Вид товара" - "Ингредиент" Появляется закладка "Ингредиент" со следующими реквизитами:
- Коэффициент выхода - процент, указывающий разницу между количеством закладки и количеством выхода ингредиента.
Наценка на ингредиент - наценка в % на ингредиент. Используется в том случае, если цена реализации рассчитывается путем 
суммирования стоимости входящих в блюда ингредиентов с определенной процентной надбавкой.
- Сезонный коэффициент 1
- Сезонный коэффициент 2
- Сезонный коэффициент 3
- Сезонный коэффициент 4
                      - процент, указывающий разницу между количеством списания и количеством закладки ингредиента (изменяется по сезонам).
- Подлежит разделке - принимает значения "Да" или "Нет" - указывает, подлежит ли товар дальнейшей разделке (документ "Акт разборки")
Из формы элемента справочника можно создать новый или открыть существующий документ "Калькуляционная карта" (закладка "Блюдо") 
и открыть подчиненный справочник "Пропорции разделки" (только для ингредиентов с реквизитом "Подлежит разделке" - "Да".
