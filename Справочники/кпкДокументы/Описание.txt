Справочник "Документы (для мобильных устройств)"

Служебный справочник, предназначенный для уникальной идентификации 
документов, которые выгружаются и загружаются из мобильных устройств.
С помощью этого справочника обработка обмена данными "Обмен данными"
определяет, был ли ранее загружен документ из МУ в центральную базу данных 
или нет, а также определяет какому документу следует подчинить загружаемый
документ для правильного формирования дерева подчиненных документов.  
Реквизит "Дата" предназначен для отображения даты создания документа 
в программе.
Поле "Агент" предназначено для вывода информации о том, какой агент создал 
документ. В реквизитах "Время создания" и "Дата создания" 
отображается соответственно время и дата создания документа в МУ, а в
реквизитах  "Широта" и "Долгота" GPS-координаты точки, где был создан 
тот или иной документ.
Реквизит "Категория", предназначен для возможности указания категории
документов, которая впоследствии может быть выгружена в МУ.
Также в поле "Категория для долга" может быть указана категория для
документа долга в МУ, который создается на основании текущего документа, 
например для документа "Реализация".
Все категории документам назначаются только непосредственно в 1С:Предприятии,
в МУ категории доступны только для просмотра и подведения итогов в журнале
документов.
