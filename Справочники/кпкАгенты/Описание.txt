Справочник "Агенты"    

Данный справочник содержит настройки, необходимые для 
организации работы с ПО «Агент Плюс». 

Все настройки логически раздлены на несколько закладок.

1. Основные            

• Мобильное устройство. Мобильное устройство, используемое торговым агентом.  

• Мобильное приложение. Мобильное приложение, используемое торговым агентом.
Выбирается одно из 2-х: "Агент Плюс 1.5" или "Агент Плюс: Мобльная торговля". 

• Ассортимент. Ассортимент товаров для выгрузки в КПК. 

• Список складов. Список складов, остатки по которым выгружаются агенту.     

• Основная фирма. Фирма, от имени которой агент оформляет документы 
  в КПК по умолчанию.  
 
• Пароль ZIP. Пароль, используемый для защиты архива.
        
2. Дополнительные
              
• Валюта агента. Валюта, используемая агентом в мобильном устройстве.     

• Основной склад.  Указанный склад будет проставляться для загружаемых 
  с КПК документов "Заявка" (если в качестве константы "Вид операции для 
  заявок" на форме "Настройка параметров для Агент+ " указано значение 
  "Заявка на склад"), а также для документов "Реализация", создаваемых на 
   основании загруажемых заявок.     

• Доступность типов цен. Данная настройка регулирует доступность выбора 
  типов цен для агента при оформлении документов. Возможна выгрузка не
  больше 32 типов цен для конкретного агента. 
   
• Передвижной склад. Передвижной склад агента (автомобиль). 
   
• Касса агента. Касса, используемая агентом при оформлении кассовых 
  документов.

• Контрагент. Указывается дополнительный контрагент, который
  будет выгружен в КПК.  Например, если агент может выступать и 
  сам как клиент, к примеру в случае формления заявок 
  на погрузку в машину (при "торговле с колес"). 
  
• Префикс нумерации документов в мобильном устройстве. Первая 
  часть префикса номеров документов, создаваемых в КПК. Вторая часть 
  префикса берется из справочника "Фирмы". 
  
3. Обмен данными     
    
Настройки для способов обмена данными:  

• Почтовый ящик. Персональный электронный адрес агента, на
  который будут отправляться файлы для МУ.     
  
• Папка обмена на FTP-сервере. Папка агента на FTP-сервере.                                                                                                       

• Папка обмена "Агент Плюс СОД". Папка агента при использовании 
  Агент+ Сервера.           

4. GPS   

• Основная карта. Карта, которая будет проставляться по умолчанию
  для агента при формировании отчета "Карта маршрута".
  
• Цвет маршрута. Цвет маршрута агента, которым будет отображаться
  маршрут и трек перемещения агента при формировании отчетов с
  использованием GPS.
