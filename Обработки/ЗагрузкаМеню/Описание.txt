Обработка "Загрузка меню" предназначена для загрузки информации из системы "R-Keeper".
При начале работы с данной обработкой требуется ввести дату смены, за которую
загружается информация. В диалоге обработки имеются два переключателя: 
"заполнять всем списком" - если галочка стоит, то в таблицу попадают все существующие
значения из системы R-Keeper, если нет, то только новые, то есть не существующие в
системе "АСТОР-Ресторан".  
"поэтапная загрузка" - при включенном переключателе становятся видимыми следующие кнопки:
"Персонал", "Модификаторы", "Причины удаления", "Валюты", "Меню", Чеки", "Марочный отчет", "Оплата чеков"
и загрузка данных производится в порядке, выбранном пользователем. Пользователь имеет
возможность контроля загрузки данных - информация о загружаемых данных отображается
в табличной части диалоговой формы обработки.     
Чтобы загрузить информацию в справочники или документы системы "АСТОР-Ресторан" после
каждой из вышеперечисленных кнопок следует нажать кнопку "Загрузить".  
При выключенном переключателе "поэтапная загрузка" вся загрузка данных осуществляется
автоматически нажатием кнопки "Загрузить".
